This section describes how to set up email notification.


== Prerequisites

* **WhizardTelemetry Notification** should have been installed and enabled.

== Steps

. Log in to the {ks_product-en} web console.

. Click the current username in the upper-right corner of the page, and then select **Notification Settings** from the dropdown list.

. On the **Notification Settings** page, under the **Email** tab, set the email notification parameters, and then click **OK**.
+
====
* Recipient Settings
+
Enter the email address, click **Add** to set multiple notification recipients.

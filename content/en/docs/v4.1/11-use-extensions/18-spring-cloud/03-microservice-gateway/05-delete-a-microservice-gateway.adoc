This section describes how to delete a microservice gateway.

// Warning
include::../../../../../_ks_components-en/admonitions/warning.adoc[]

Deleting the microservice gateway will cause the microservice's business to be interrupted. Please proceed with caution.

include::../../../../../_ks_components-en/admonitions/admonEnd.adoc[]

== Prerequisites

* **KubeSphere Spring Cloud** should have been installed and enabled.

本节介绍如何登录{ks_product-en} Web 控制台并进入集群。


== 前提条件

* 您需要获取集群成员的用户名和密码。

include::../../../_custom-en/workspaceManagement/webConsole/webConsole-prer-logIn.adoc[]


== 操作步骤

. 打开 Web 浏览器，在地址栏中输⼊{ks_product-en} Web 控制台的 IP 地址和端⼝号。 
. 在{ks_product-en}登录⻚⾯，输⼊⽤⼾信息然后点击登录。
. 点击**集群管理**。
. 如果已有多个集群，在**集群管理**页面点击一个集群的名称进入集群。

* 对于**TLS 信息**类型的保密字典，设置以下参数：
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|证书
|TLS 证书。

|私钥
|TLS 私钥。
|===
--

* 对于**镜像服务信息**类型的保密字典，设置以下参数：
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|镜像服务地址
|镜像服务的地址。点击下拉列表可选择 HTTP 或 HTTPS 协议。

|用户名
|用于访问镜像服务的用户名。

|邮箱
|用于访问镜像服务的用户的邮箱。

|密码
|用于访问镜像服务的密码。

* 点击**验证**可检查密码是否正确。

* 点击image:/images/ks-qkcp/zh/icons/eye.svg[eye,18,18]/image:/images/ks-qkcp/zh/icons/eye-closed.svg[eye-closed,18,18]可显示/隐藏密码明文。

|数据
|用于访问镜像服务的键值对数据。
|===
--

* 对于**用户名和密码**类型的保密字典，设置以下参数：
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|用户名
|用户自定义的用户名。

|密码
|用户自定义的密码。点击image:/images/ks-qkcp/zh/icons/eye.svg[eye,18,18]/image:/images/ks-qkcp/zh/icons/eye-closed.svg[eye-closed,18,18]可显示/隐藏密码明文。
|===

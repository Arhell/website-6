本节介绍如何管理日志接收器。

//note
[.admon.note,cols="a"]
|===
|说明

|
安装并启用 WhizardTelemetry 日志管理、WhizardTelemetry 事件管理、WhizardTelemetry 审计管理、WhizardTelemetry 通知管理这四个扩展组件中的任意一个，集群左侧导航栏的**集群设置**菜单下都会显示**日志接收器**选项，**日志接收器**页面则会显示对应的页签。

|===

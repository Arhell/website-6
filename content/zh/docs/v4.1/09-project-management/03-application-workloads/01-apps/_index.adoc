本节介绍如何管理应用。

在{ks_product_both}平台，应用特指由一个或多个工作负载、服务、应用路由等资源组成的业务程序。

{ks_product_right}平台目前支持创建基于模板的应用，即通过已有的应用模板创建应用。创建应用所使用的应用模板可以是已发布到应用商店的应用模板、上传到企业空间的应用模板、或第三方应用仓库中的应用模板。
